# 🎬 Movie App 2024

React JS 기초 과정 2024를 통해 개발한 **Movie App 2024**입니다.

## 🌐 라이브 데모

🔗 [movie-rating-web.vercel.app](https://movie-rating-web.vercel.app)

## 🛠️ 사용된 기술 스택
- ⚛️ React.js          
- 🚀 Vercel             
- 🎨 CSS               
- 🌐 Axios

## 🎥 기능
- 🍿 인기 영화 목록 보기
- ⭐️ 영화 평점 및 리뷰 보기
- 📝 영화에 대한 상세 정보 확인 (제목, 개봉 연도, 감독, 출연진 등)
